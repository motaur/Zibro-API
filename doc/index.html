ZIBRO-API

Documentation

Format: JSON. No Authentication

Events group

/api/events returns all events with all fields, with images arrays.

/api/events/:id returns a specific event.

/api/eventsByOrganizer/:id returns events by organizer id with all fields and images arrays

/api/eventsByStatus/:id returns events by status id with all fields and images arrays

/api/events/ Returns posted event with all fields
Fields are required:
- title (string minimum 3 characters), organizerid (string), date (timestamp)
Fields are optional:
- description (string) datedescription (string) locationname (string) type (integer, 1 is minimum allowed, default value: 1) status (integer, 1 is minimum allowed, default value: 1) locx (float) locy (float) price (float, 0 is minimum allowed, default value: 0)
Any other fields are not allowed

/api/events/:id Returns edited event with all fields
Parameter id of event is required after "/" Fields are optional:
- title (string minimum 3 characters), organizerid (string) date (timestamp) description (string) datedescription (string) locationname (string) type (integer, 1 is minimum allowed, default value: 1) status (integer, 1 is minimum allowed, default value: 1) locx (float) locy (float) price (float, 0 is minimum allowed, default value: 0)
any other fields are not allowed
